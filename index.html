Configure .gitconfig

Add your name and email at a minimum

git config --global user.name "Rob Hernandez"
git config --global user.email "rob@sarcasticadmin.com"

This adds configuration to your user's ~/.gitconfig

~$ cat ~/.gitconfig
[user]
name  = Rob Hernandez
email = rob@sarcasticadmin.com
Without a .gitconfig
